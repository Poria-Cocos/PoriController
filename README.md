# PoriController
机器人、无人机、遥控车通用遥控方案。PCB、代码全部开源。遥控资源：2个摇杆、8个正面按键、4个三档开关、4个肩键、2个旋转编码器、1个串口屏幕。

## 各资源链接
- [项目主页](https://poria-cocos.github.io/aritcles/4fc11e3.html)：项目介绍、材料清单
- [代码GitHub仓库](https://github.com/poria-cocos/PoriController)：固件代码
- [原理图和PCB](https://oshwhub.com/poria_cocos/tong-yong-yao-kong-_poricontroller)：遥控主板、电池管理板

## 仓库文件说明
- `POC_code`：遥控器固件代码

## 遥控传输数据包
> 以下为 POC_code 中无线传输的数据包，可根据实际使用的无线模块及需求修改

**共 26 字节**

| 字节 | 内容 | 说明 |
|------|------|------|
| 0    | 00   | 无线模块地址高8位 |
| 1    | AA   | 无线模块地址低8位 |
| 2    | 01   | 无线模块信道 |
| 3    | AA   | 包头1 |
| 4    | AA   | 包头2 |
| 5    |      | 远程急停控制位 |
| 6    | LX   | 左摇杆X |
| 7    | LY   | 左摇杆Y |
| 8    | RX   | 右摇杆X |
| 9    | RY   | 右摇杆Y |
| 10(87654321) 11(8765)  | SW0~11 | 按键0~11（按下为1） |
| 11(4321) 12(8765)   | Toggle 1A 1B 2A 2B 3A 3B 4A 4B | 钮子开关1~4 |
| 12(4321) 13(8765)   | Encoder LSW RSW, Rocker LSW RSW, Encoder LA+ LB- RA+ RB- | 编码器按键，摇杆按键，编码器旋转触发 |
| 13(4321)   |      | 预留 |
| 14   | 页面id(高4位), 控件id(低4位) | 串口屏页面及控件 |
| 15   |      | 控件值低字节 |
| 16   |      | 控件值高字节 |
| 17~22|      | 预留 |
| 23   | 0~255 | 计数值 |
| 24   | AA   | 包尾1 |
| 25   | AA   | 包尾2 |
